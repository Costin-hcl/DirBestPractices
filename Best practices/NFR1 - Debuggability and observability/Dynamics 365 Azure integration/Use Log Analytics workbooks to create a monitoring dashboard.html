Using Log Analytics workbooks to create a monitoring dashboard

Log Analytics workbooks can query data from multiple sources and display it through rich visualization reports.

The data sources available are:

  • Logs
  • Metrics
  • Azure Resource Graph
  • Alerts (Preview)
  • Workload Health
  • Azure Resource Health
  • Azure Data Explorer


Aggregate data can then be viewed using tiles, honey comb diagrams, charts or simply tables.

The data can be selected using Kusto queries.

As an example, to get all the failed Logic App runs in a resource group for the last 24 hours, we can run the query below:

AzureDiagnostics
| where Category == 'WorkflowRuntime' and TimeGenerated > now() - 24h 
| where not(status_s  in ('Running', 'Skipped', 'Succeeded'))
