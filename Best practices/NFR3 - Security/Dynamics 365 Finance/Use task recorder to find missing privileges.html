Using Task Recorder to find missing privileges

Each user should be assigned a role that enables them to perform all the tasks that they need to and nothing more.

Sometimes, it can happen that a duty is missed for a user. To prevent that, we can use a standard feature in D365F&SCM that checks whether a user has all the roles for performing a set of tasks specified in a task recording.

See documentation in this ebook.

The key steps to perform are:

  • Capture task recording of business processes
  • Upload the task recording for analysis using System administration > Security > Security diagnostics for task recordings
  • Verify the missing privileges or duties for a specific user.
