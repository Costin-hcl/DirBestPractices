Using Azure Cost Management

Azure Cost Management and Billing offers out of the box reports for monitoring spend summarized by various useful measures:

  • type of resource
  • resource group
  • region

By grouping each set of systems into one resource group (e.g. all resources for user acceptance testing in Contoso-RG-UAT), we can forecast the production costs, assuming tests with realistic production volumes are carried out.

It's good to discuss the billing reports periodically with the end-user.
