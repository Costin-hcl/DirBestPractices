Actions in Logic Apps should have meaningful names

Frequently used actions like "Parse JSON", "HTTP" or "Set variable" are likely to appear multiple times. As actions need to have unique names, the default behavior of the visual editor is to add a number at the end (e.g. "Set variable 2"). This makes the Logic Apps harder to read and should be avoided.
