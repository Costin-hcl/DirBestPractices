Are buttons with different functionality using similar icons or colors

Buttons have several ways of suggesting function to the user:

  • icon
  • color
  • size
  • text

These elements need to be consistent and also intuitive. Here are some best practices for each:

Icons

  • Use standard icons like Font Awesome, Bootstrap Icons for web apps or the Segoe UI font for Windows apps. These ensure common metaphors and symbols that people are used to from mobile phones and web pages.
  • Do not use the same icon for buttons with a different function.

Color

  • Use colors consistently and be mindful of the sentiment they suggest (red-amber-green).
  • Don't use too many colors as it may become confusing. A uniform UI with only a few elements using striking colors like red or green tends to be more intuitive.

Size

  • Use similar sizes for buttons with equally important functions.
  • Make important buttons (e.g. Submit, Buy, Environment down incident) larger.

Text

  • Be aware that the text of a button may not be read by the user, as icons and color are easier to spot.
  • Don't use labels that are too short and don't explain the functionality.
  • Don't use labels that are too long so users avoid reading them
