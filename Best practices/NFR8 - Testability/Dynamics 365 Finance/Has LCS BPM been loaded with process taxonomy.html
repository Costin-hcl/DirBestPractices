Loading process taxonomy in LCS BPM

Loading the full list of processes that need to be supported by the implemented system in the LCS Business Process Modeler (BPM) makes testing easier as it defines the full scope and also provides a hierarchical grouping of scenarios to be implemented and tested.

The LCS BPM library can be synchronized with Azure DevOps, which will create the hierarchy of Epics, Features and User stories. Test cases can then be linked to the user stories in order to track the completeness and status of testing against business requirements.
