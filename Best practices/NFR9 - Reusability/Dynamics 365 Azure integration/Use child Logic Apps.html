Using child Logic Apps

Just like in C# source code it's better to move code sections that appear multiple times into separate functions, in Logic Apps, repeated sequences of actions can be moved to a child Logic App.

Using child Logic Apps for repeated actions also makes the whole solution easier to test. Another advantage is avoiding nested loops, which will run sequentially rather than in parallel unless move to a child Logic App.
